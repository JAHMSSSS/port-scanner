# Port Scanner

A simple port scanner tool with a graphical user interface.

## Features

- Scans a range of ports on a given host.
- Displays open ports and their associated protocols.
- Easy-to-use GUI.

## Requirements

- Python 3
- tkinter
- pyfiglet
